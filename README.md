# DSA Practice in Java

This repository contains my daily practice problems in Data Structures & Algorithms (DSA), implemented in Java.  
Each folder represents a day’s work with source code and notes.

## Structure
- **Day1** → Majority Element
- **Day2** → Kadane's Algorithm, Two Sum
- **Day3** → Array Sort Merge, Missing Repeated Values
- **Day4** → Single Number, Stock Profit
- **Day5** → Power Function (pow)
- **Day6** → Container With Most Water
- **Day7** → Sort Colours

## Notes
- `.class` files are ignored (only source `.java` files are tracked).
- Each folder has a `README.md` with problem statements and approaches.
